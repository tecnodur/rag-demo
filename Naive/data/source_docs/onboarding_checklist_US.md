# US Employee Onboarding Checklist

## Purpose
This checklist standardizes onboarding for new US hires across payroll, benefits enrollment, tax setup, device handoff, and account activation. It is intended to reduce delays in first-week productivity and ensure baseline compliance.

## Pre-Start Tasks (Manager and HR)
Complete prior to start date:
- Confirm signed offer and start date in HRIS
- Validate work state and employing entity
- Submit onboarding request with role, department, and manager
- Assign first-week orientation agenda
- Confirm equipment request and shipping or pickup details

## HR, Tax, and Benefits Setup
Required for US employees:
- Initiate federal and state tax setup (including W-4 workflow)
- Complete I-9 process and employment eligibility steps
- Enroll employee in benefits selection window
- Confirm payroll direct deposit details
- Provide mandatory workplace policy acknowledgments

## IT Provisioning and Access
IT Enablement should complete:
- Create enterprise identity account and email access
- Provision collaboration tools and ticketing login
- Configure endpoint security and device compliance tools
- Grant role-based application permissions
- Provide account activation instructions

## Password and Account Activation
On first login, the new employee must:
- Set initial password and complete authentication enrollment
- Confirm access to email, collaboration tools, and HR systems
- Validate backup authentication method for account recovery
- Review security guidance for remote login and device use

Note: This checklist includes password setup tasks but does not define regional password policy standards.

## First Week Completion
By the end of week one:
- Verify payroll status and tax workflow completion
- Confirm benefits enrollment or waiver submission
- Resolve pending access tickets
- Capture onboarding completion in HR tracker
